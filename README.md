# PrimeAxis Technologies Website

A modern, responsive React website for PrimeAxis Technologies — a technology solutions company specializing in Zoho ERP & CRM, custom software, mobile apps, AI solutions, and digital marketing.

## Tech Stack

- **React 19** with Vite
- **Tailwind CSS v4** for styling
- Fully responsive, mobile-first design

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Sections

- Hero with CTA buttons
- About Us (Mission & Vision)
- Services (6 service cards)
- Why Choose PrimeAxis
- Industries We Serve
- Development Process (6 steps)
- Technologies We Use
- Portfolio (placeholder projects)
- Call to Action
- Contact form & information
- Footer

## Customization

- Update portfolio items in `src/components/Portfolio.jsx`
- Add real project images and case studies as they become available
- Contact form opens the user's email client with pre-filled details
